================================================================================
                          WHAT IS THIS GAME?
================================================================================

Blokus is a strategy board game where players try to place all their pieces
on the board. This is a 2-player online multiplayer version.

FEATURES:
  ‚úì 2 players (Blue vs Red)
  ‚úì Online multiplayer with Firebase
  ‚úì 14√ó14 game board
  ‚úì 21 unique piece shapes per player
  ‚úì Real-time synchronization
  ‚úì Room-based matchmaking (6-digit codes)
  ‚úì Mobile responsive
  ‚úì No installation required

RULES:
  1. First piece must cover your starting corner (‚òÖ)
  2. Other pieces must touch corner-to-corner with your pieces
  3. Cannot touch edge-to-edge with your own pieces
  4. Take turns placing pieces
  5. Game ends when both players skip
  6. Highest score wins!

================================================================================
                         TROUBLESHOOTING
================================================================================

PROBLEM: Can't create room
SOLUTION: Check that Firebase config is correct in index.html (line ~698)

PROBLEM: Room not found when joining
SOLUTION: Verify the 6-digit code is correct, room may have expired

PROBLEM: Game not syncing between players
SOLUTION: Check internet connection, verify Firebase rules allow read/write

For more help, see FIREBASE_SETUP.md ‚Üí Troubleshooting section

                           QUICK REFERENCE
================================================================================

GAME CONTROLS:
  Rotate ......... Rotate selected piece 90¬∞ clockwise
  Flip ........... Mirror selected piece horizontally
  Deselect ....... Cancel piece selection
  Skip Turn ...... Pass your turn
  Leave Game ..... Exit and return to lobby

KEYBOARD:
  None - all mouse/touch based

PLAYER COLORS:
  Blue ........... Player 1 (host, starts at top-left)
  Red ............ Player 2 (guest, starts at bottom-right)

STARTING CORNERS:
  Blue: (0,0) - top-left
  Red: (13,13) - bottom-right

SCORING:
  Base: -1 point per unused square
  Bonus: +15 if all pieces used
  Bonus: +5 if last piece was the 1-square piece
  Winner: Highest score
