DATABASE STRUCTURE:

    gameState/
      board: 14x14 array

      currentPlayerIndex: 0 or 1
      passCount: 0-2

================================================================================
                            SETUP PROCESS
================================================================================

TIME REQUIRED: ~5 minutes total

STEP 1: Firebase Project (2 minutes)
  → Create project at console.firebase.google.com
  → Enable Realtime Database
  → Set rules to test mode

STEP 2: Get Configuration (1 minute)
  → Project Settings → Your apps → Web app
  → Copy firebaseConfig object

STEP 3: Update Code (1 minute)
  → Open index.html
  → Find line ~698 (FIREBASE CONFIGURATION)
  → Paste your config values

STEP 4: Test (1 minute)
  → Open index.html in browser
  → Create room
  → Join from another browser/tab
  → Play!

================================================================================
                          USAGE INSTRUCTIONS
================================================================================

PLAYER 1 (HOST):
1. Open index.html in browser
2. Click "Create Room"
3. Share the 6-digit room code
4. Wait for opponent to join
5. Game starts automatically
6. Play as Blue (goes first)

PLAYER 2 (GUEST):
1. Open same index.html file
2. Click "Join Room"
3. Enter the 6-digit code
4. Click "Join"
5. Game starts automatically
6. Play as Red (goes second)

GAMEPLAY:
- Select piece from right panel
- Rotate/Flip as needed
- Click board to place
- First piece must cover your starting corner (★)
- Other pieces must touch corner-to-corner
- Cannot touch edge-to-edge with your own pieces
- Take turns until no more moves
- Both skip = game ends
- Highest score wins!
